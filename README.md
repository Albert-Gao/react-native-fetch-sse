# react-native-fetch-sse

consume text-stream response based on standard fetch.

## Installation

```sh
npm install react-native-fetch-sse
```

## Usage

```js
import { multiply } from 'react-native-fetch-sse';

// ...

const result = await multiply(3, 7);
```
